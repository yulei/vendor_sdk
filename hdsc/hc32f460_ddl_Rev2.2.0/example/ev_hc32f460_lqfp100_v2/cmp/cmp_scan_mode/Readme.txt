平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示CMP的扫描比较功能。
说明：
样例连续扫描检测INP1和INP2，通过VCOUT引脚，查看检测状态。

================================================================================
使用步骤
================================================================================
1）CMP2_INP1(PA4)与ADC电位器输出(PC0)连接，VCOUT2(PB13)与示波器连接
2）打开工程编译并全速运行
3）调节ADC电位器输出为最小值
4）CMP2_INP2(PA5)接3.3V，观察示波器，显示连续脉冲方波，表示INP2电压输入高于INM电压
5）CMP2_INP2(PA5)接低电平，示波器显示低电平，表示INP2电压输入低于INM电压
6）CMP2_INP2(PA5)接低电平，调节ADC电位器输出为最大值
7）示波器显示连续脉冲方波，表示INP1电压输入高于INM电压
8）CMP2_INP2(PA5)接低电平，调节ADC电位器输出为最小值
9）示波器显示低电平，表示INP1电压输入低于INM电压
10）重复步骤3）~步骤9）
