平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU USB外设实现的的USB host CDC功能。

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行，使用板载J1串口查看运行输出信息。
2）  连接OTG转接线，再接上CDC ACM设备
     (可以使用另一块开发板烧录 usb_dev_cdc 样例获得该设备)。
3）  成功识别CDC后led闪烁。
4）  根据J1串口输出提示：
     短按SW2测试发送数据，短按SW3测试修改CDC ACM设备的波特率(需设备支持该功能)。
     当接收到设备发来的数据，J1串口会进行显示输出。
